name: cc-session
description: Resume, close, and review Google Antigravity standalone app sessions using cc-session filtered handoffs and evidence IDs.
---

This wrapper targets the Google Antigravity standalone app brain store, not the Antigravity IDE-like storage layout.

Shared workflows are installed beside this skill:

- `common/resume-session.md`
- `common/close-session.md`
- `common/review-history.md`

Use `cc-session` CLI or MCP tools instead of reading raw Antigravity transcript JSONL directly.
